# Hongsu Portfolio

Technical Artist portfolio website.

## Stack
- Next.js
- TypeScript
- React

## Goals
- Extensible landing architecture
- Future shader/effect showcase
- Clear separation between content and rendering

## Routes
- `/`
- `/projects/[slug]`
- `/about`
- `/contact`

## Development
1. Install Node.js 20+ and npm
2. Run `npm install`
3. Run `npm run dev`
